How to Use Stimulus Presentation Under Pyff

StimulusPresentation is a feedback application developed under Pyff (Pythonic Feedback Framework).

Initiate the Parallel Port

To utilize Pyff with the Neuroscan SynAmps system you must first initiate the parallel port by opening the program LPT Port Test Utility

  1. Open the LPT Port Test Utility from the Start Menu
  2. Select _LPT X address from dropdown menu at the top of the program
  3. Enter the hardware address 0378 in the relevant field and press enter

Your parallel port should now be initiated and you can close the Port Test Utility.
